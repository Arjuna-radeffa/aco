Infrastruktur Pengiriman
💬

WhatsApp Business API

Provider: Fonnte / WABLAS / Official Meta WABA
Semua WA menggunakan message template yang sudah disetujui Meta sebelum dikirim.

📧

Email Transactional

Provider: Mailgun / SendGrid / AWS SES
Sender: noreply@harmonivita.id
Reply-to: hello@harmonivita.id

⏱

Scheduled Jobs

Cron job harian pukul 07:00 WIB untuk reminder terjadwal (perpanjangan, kontrol). Notifikasi transaksional dikirim real-time via queue (maks. 5 menit).

🔁

Retry Policy

WA gagal → retry 1x setelah 15 menit. Jika masih gagal → fallback email (untuk notif kritis). Log semua status pengiriman.
